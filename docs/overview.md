# DevForge Overview

DevForge CLI is a production-focused scaffolding tool for JavaScript and TypeScript teams that want more than a one-shot starter template.

## Project Flow

1. Inspect the local environment.
2. Collect intent, tooling, state management, testing, and architecture choices.
3. Normalize invalid or risky combinations.
4. Generate code, documentation, and AI rules.
5. Optionally install dependencies and initialize git.

## Product Goals

- reduce project setup time without hiding architecture decisions
- encode compatibility rules so teams avoid broken combinations
- generate docs and repo hygiene alongside code
- create AI rules and skills guidance aligned with the selected stack

## Current Scope

- frontend, backend, fullstack, microfrontend, extension, and CLI scaffolding
- testing setup prompts and starter configs
- ESLint, Prettier, and Husky profile selection
- GitHub-ready repository structure and contribution files
